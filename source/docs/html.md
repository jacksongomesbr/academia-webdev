[HTML (HyperText Markup Language)](http://www.w3.org/TR/html5/) é, basicamente, a linguagem base e universal para conteúdo na internet. 


``` html
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Title of the document</title>
</head>

<body>
Content of the document......
</body>

</html>
```
